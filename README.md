# Новиков Максим ДПИ23-1

Информационно-справочная система логистической компании

## Состав

- `logistics-server` – серверная часть
- `logistics-client` – клиентская часть

## Требования

- Java 17
- Maven
- PostgreSQL

## Запуск

1. Запустить сервер: `cd logistics-server && mvn spring-boot:run`
2. Запустить клиент: `cd logistics-client && mvn javafx:run` (или через IDE)
